# Customer, Product and Profitability Performance Analysis

## Project Overview
This project analyzes customer profitability, product performance, discount impact, and market profitability using supply chain data.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab
- GitHub

## Key Findings
- Identified top profitable customers
- Analyzed product category performance
- Evaluated discount impact on margins
- Compared market and regional profitability

## Conclusion
The project provides business intelligence insights that help organizations improve profitability and decision-making.
